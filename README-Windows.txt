LINE Sticker Capture App - Windows用ユーザーガイド
======================================================

🖼️  LINE STOREのスタンプを個別にキャプチャするデスクトップアプリケーション

--------
1. システム要件
2. 初期セットアップ

5. トラブルシューティング

1. システム要件
---------------
✅ Windows 10/11 (64bit推奨)
✅ Node.js 18.0以上 (https://nodejs.org/)
✅ インターネット接続
✅ 最低2GBの空きディスク容量

2. 初期セットアップ
------------------
📦 自動セットアップ（推奨）:
   1. setup.bat をダブルクリック
   2. 画面の指示に従って進める
   3. セットアップ完了後、デスクトップショートカットが作成されます

📦 手動セットアップ:
   a) Node.jsをインストール
      - https://nodejs.org/ から「LTS」版をダウンロード
      - インストーラーを実行
      - 「npm」も一緒にインストールされることを確認

   b) PowerShell実行ポリシーの設定
      - PowerShellを管理者権限で開く
      - 以下のコマンドを実行:
        Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

   c) デスクトップショートカットの作成
      - create-shortcut.ps1 を右クリック
      - 「PowerShellで実行」を選択

3. アプリの起動方法
------------------
🚀 方法1: デスクトップショートカット（推奨）
   - デスクトップの「LINE Sticker Capture」をダブルクリック

🚀 方法2: バッチファイル直接実行
   - start-app.bat をダブルクリック

4. 使用方法
-----------
📋 基本的な使用手順:

   1. アプリケーションの起動
      - デスクトップショートカットをダブルクリック
      - 初回起動時は依存関係のダウンロードで数分かかります

   2. URLの入力
      - LINE STOREのスタンプページのURLをコピー
      - アプリの「URL」フィールドに貼り付け
      - 例: https://store.line.me/stickershop/product/1234567/ja

   3. キャプチャ開始
      - 「開始」ボタンをクリック
      - ブラウザが自動で開きます

   4. ポップアップの手動対応
      - ブラウザでポップアップ広告が表示された場合
      - 手動でポップアップを閉じてください
      - アプリで「完了」ボタンをクリック

   5. 自動キャプチャ
      - ページ内のすべてのスタンプが自動でキャプチャされます
      - 進行状況がアプリに表示されます

   6. ファイル保存
      - キャプチャされた画像は「captures」フォルダに保存されます
      - ファイル名: [スタンプセット名]_[番号].png

📁 保存場所:
   - キャプチャファイル: [アプリフォルダ]\app\captures\
   - ログファイル: [アプリフォルダ]\app\logs\

5. トラブルシューティング
------------------------
❌ 「Node.js がインストールされていません」
   → https://nodejs.org/ からNode.js LTS版をインストール

❌ 「ショートカット作成に失敗しました」
   → PowerShellの実行ポリシーを設定:
     Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

❌ 「依存関係のインストールに失敗しました」
   → 1. インターネット接続を確認
     2. ウイルス対策ソフトを一時的に無効化
     3. 管理者権限でstart-app.batを実行

❌ ブラウザが起動しない
   → Chromeブラウザがインストールされているかを確認

❌ スタンプがキャプチャされない
   → 1. ポップアップを完全に閉じているかを確認
     2. ページが完全に読み込まれるまで待機
     3. URL形式が正しいかを確認

❌ アプリが応答しない
   → 1. コマンドプロンプトを閉じて再起動
     2. Ctrl+Cでプロセスを終了
     3. PCを再起動

🔧 ログファイルの確認:
   問題が発生した場合、以下のファイルを確認:
   - app\logs\app.log
   - setup.log (セットアップ時の問題)

   - 個人情報の収集は行いません
